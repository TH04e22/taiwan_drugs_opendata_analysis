#全台毒品犯罪資料分析#
***
藉由Weka數據分析軟體分析內政部所提供之開放數據，搭配Tableau的視覺化效果，呈現以DBSCAN處理後的數據，以便觀察各項趨勢，提供警方建議依此分析結果根據各區狀況加強毒品防範措施。

* 分析軟體: Weka
* 可視化軟體: Tableau
